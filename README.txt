图书管理系统 — 使用说明

一、直接运行
将 BookManageMFC.exe_ 重命名为 BookManageMFC.exe，双击运行。
登录账号 admin，密码 book2026。
（PTA 平台不允许上传 .exe 文件，故将后缀改为 .exe_，请改回后使用。）

二、VS 编译
1. Visual Studio 2022/2026 打开 BookManageMFC.sln
2. 如果编译报错 MSB8020（找不到 v145 工具集）：
   右键项目 → 属性 → 配置属性 → 常规 → 平台工具集 → 改为 v143
   （VS 2022 自带 v143，VS 2026 自带 v145）
3. 确保已安装 "C++ MFC for v14xx build tools"（VS Installer 中勾选）
4. 配置选择 Release x64，生成解决方案

三、文档

本项目在开发过程中先后制作了两版文档，体现了持续改进的努力：

1. 大程报告_终版.pdf —— 最终实验报告（请优先查看）
   教师提供了统一的实验报告模板（含功能完成自我评价表、目录、6 章正文结构、
   参考文献等要求）后，我们按照模板重新撰写。内容全面覆盖：大程序简介、需求
   分析（业务/功能/数据/性能）、类库已有功能分析（MFC 框架分析）、新设计类
   功能说明（自定义类详述）、部署运行与使用说明、参考文献资料（10 条，均已
   交叉验证确保真实可查）。含封面（学院/专业/学号/姓名/指导教师/日期）、
   功能完成自我评价表、UML 类图、5 张运行截图。

2. 使用说明.pdf —— 面向用户/教师的操作手册
   包含运行方式、界面布局、各功能按钮的操作说明和异常处理指南。

3. 设计文档.pdf —— 早期版本的设计文档（教师提供模板前制作）
   涵盖架构设计、类接口说明、评分对照等，可作为大程报告的补充参考。
